# Sphere-to-app story

This slide is controlled with Space, not the arrow keys.

Start with the full sphere card and explain the rule from the web app: every €25 lights one athlete. Press Space once to bring in the phone preview and show the real hosted app inside the phone. Press Space again to send the phone away and let the sphere react as if a $50 donation happened.

The $50 moment lights two athletes, so the sphere plays two sequential meteor/wave pulses. The newly added people appear in black, matching the web app's sponsored-person treatment.
